Instructions to build and merge bin files for ESP32C6 SPI wifi example project (esp-hosted-mcu package):
- Download source: https://github.com/espressif/esp-hosted-mcu
- Source folder: /esp-hosted-mcu/slave
- Docs folder: /esp-hosted-mcu/docs
- Build:
	- Run ESP-IDF 5.3 PowerShell
	- Change to the source folder above
	- Run idf.py set-target esp32c6 to set target ESP chip
	- Run idf.py menuconfig to make changes to configuration
	(Main menu -> Example Configuration -> xxx to change SPI pins and DataReady and Handshake pins)
(
CONFIG_ESP_SPI_HSPI_GPIO_MOSI=12
CONFIG_ESP_SPI_HSPI_GPIO_MISO=13
CONFIG_ESP_SPI_HSPI_GPIO_CLK=7
CONFIG_ESP_SPI_HSPI_GPIO_CS=15
CONFIG_ESP_SPI_GPIO_MOSI=12
CONFIG_ESP_SPI_GPIO_MISO=13
CONFIG_ESP_SPI_GPIO_CLK=7
CONFIG_ESP_SPI_GPIO_CS=15
CONFIG_ESP_SPI_GPIO_HANDSHAKE=14
CONFIG_ESP_SPI_GPIO_DATA_READY=6
CONFIG_ESP_SPI_GPIO_RESET=-1
)	
	- Run idf.py build to compile and build the images
- Merge and convert bin files to a hex file or a raw file:
	- Change to folder build
	- Run the command: esptool.py --chip ESP32C6 merge_bin --format hex -o esp32c6-flash.hex --flash_mode dio --flash_size 4MB 0x0000 bootloader/bootloader.bin 0x8000 partition_table/partition-table.bin 0xd000 ota_data_initial.bin 0x10000 network_adapter.bin
or	- Run the command: esptool.py --chip ESP32C6 merge_bin --format raw -o esp32c6-flash.bin --flash_mode dio --flash_size 4MB 0x0000 bootloader/bootloader.bin 0x8000 partition_table/partition-table.bin 0xd000 ota_data_initial.bin 0x10000 network_adapter.bin
(
or	- Run the command: esptool.py --chip ESP32C6 merge_bin --format raw -o esp32c6-flash.bin --flash_mode dio --flash_size 4MB 0x0000 bootloader/bootloader.bin 0x8000 partition_table/partition-table.bin 0xd000 ota_data_initial.bin 0x1e000 at_customize.bin 0x1f000 customized_partitions\mfg_nvs.bin 0x60000 esp-at.bin
)
- Get the MD5 of the merged file:
	- Run the command:  Get-FileHash esp32c6-flash.bin -Algorithm MD5
- Save the MD5 checksum to the file esp32c6-flash.md5
